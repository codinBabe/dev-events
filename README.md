# Dev Events Application

Dev Events is a comprehensive web application designed to facilitate the creation, management, and discovery of tech events. It provides a user-friendly interface for event organizers to manage their events and for attendees to find and register for events of interest.

## Key Features

- **Event Creation:** Organizers can easily create and publish new events with all necessary details.
- **Event Management:** Organizers can edit or delete their events, as well as manage registrations.
- **Admin Authentication:** Admins can log in, browse events, and manage all aspects of the event lifecycle.
- **Responsive Design:** The application is fully responsive and works on all devices.

## Tech Stack

- **Frontend:** Built with Next.js and React for a modern, dynamic user experience.
- **Backend:** Next.js server actions handle the server-side logic, with MongoDB as the database.
- **Authentication:** Admin authentication is implemented to secure the dashboard and event management features.

# Admin Dashboard with CRUD Operations

## Overview

This implementation provides a complete admin dashboard for managing events in the Dev Events application. The dashboard includes full CRUD (Create, Read, Update, Delete) functionality with a clean, responsive UI.

### Features

- **Event Listing:** View all events in a paginated table with essential details.
- **Create Event:** Add new events using a form with validation.
- **Edit Event:** Update existing event details through an intuitive interface.
- **Delete Event:** Remove events with confirmation prompts to prevent accidental deletions.
- **Responsive Design:** The dashboard is fully responsive and works on all devices.

## Getting Started with Admin Dashboard

To access and use the admin dashboard, follow these steps:

1. Ensure you have the Dev Events application running locally.
2. Navigate to the admin dashboard URL, typically `http://localhost:3000/admin/login`.
3. Log in with your admin credentials to access the dashboard features.

## Getting Started

To get started with the Dev Events application, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/codinBabe/dev-events.git
   cd dev-events
   ```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to see the application in action.
